# VALE VOICE LAB — Fase 01

Build **VVL-1.0.0-F01** · 21/07/2026 · 17:23 BRT

Base visual e arquitetural da plataforma profissional de avaliação vocal. Para publicar, envie o conteúdo desta pasta a um repositório e habilite **GitHub Pages** na raiz do branch desejado.

## Estrutura

- `index.html` — tela inicial.
- `css/` — design responsivo.
- `js/` — interação e registro do PWA.
- `assets/` — ícones e futuros recursos visuais.
- `docs/` — auditorias da fase.

Os arquivos de áudio e os algoritmos de análise serão incluídos nas fases seguintes.
