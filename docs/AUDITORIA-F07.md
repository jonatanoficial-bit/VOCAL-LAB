# Auditoria — Fase 07: sustentação e intensidade

**Build:** VVL-1.0.0-F10  
**Data:** 22/07/2026 · 12:10 BRT

- O teste só coleta intensidade durante sinais vocais válidos.
- O tempo exibido é de sessão; a estabilidade usa a dispersão em dB das amostras disponíveis.
- O resultado é um indicador de consistência sonora, não uma medida clínica de capacidade respiratória.
